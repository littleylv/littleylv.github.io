本文用于记录日常工作中用到或了解到的一些第三方开源库。不定期更新。

序

Awesome 系列资源是 Github 上网友发起的 Awesome 项目资源列表，是一个汇总了优秀工具资源的大集合，并由 GitHub 社区用户持续维护和更新。如：awesome-php、awesome-python等。更有整合了所有 Awesome 列表的 Awesome 列表：Awesome。

本文用于记录日常工作中用到或了解到的一些第三方开源库。不定期更新。

后端

CakePHP

CakePHP: The Rapid Development Framework for PHP - Official Repository http://cakephp.org

Spout

Read and write spreadsheet files (CSV, XLSX and ODS), in a fast and scalable way.

PHP IMAP

PHP class to access mailbox by POP3/IMAP/NNTP using IMAP extension.

html-parser

php html parser，类似与PHP Simple HTML DOM Parser，但是比它快好几倍。

前端

Bootstrap

The most popular HTML, CSS, and JavaScript framework for developing responsive, mobile first projects on the web. http://getbootstrap.com

BootstrapFormHelpers

BootstrapFormHelpers is a set of javascript to help you create user friendly forms. http://bootstrapformhelpers.com

FormValidation

The best @jquery plugin to validate form fields. Designed to use with Bootstrap + Zurb Foundation + Pure + SemanticUI + UIKit + Your own frameworks. http://formvalidation.io

注：收费版本。50美刀。

Bootstrap Tags Input

Bootstrap Tags Input is a jQuery plugin providing a Twitter Bootstrap user interface for managing tags. http://bootstrap-tagsinput.github.io/bootstrap-tagsinput/examples/

bootstrap-fileinput

An enhanced HTML 5 file input for Bootstrap 3.x with file preview, multiple selection, and more features. http://plugins.krajee.com/file-input

Swiper

Most modern mobile touch slider with hardware accelerated transitions. http://idangero.us/swiper/

hash.js

hash.js is a super tiny framework to handle your location.hash.

pace

Automatically add a progress bar to your site. #hubspot-open-source http://github.hubspot.com/pace/docs/welcome

Waterfall

jQuery waterfall plugin,like Pinterest. http://wlog.cn/waterfall/

jQuery.dotdotdot

jQuery.dotdotdot, advanced cross-browser ellipsis for multiple line content. http://dotdotdot.frebsite.nl/

Fabric.js

Javascript Canvas Library, SVG-to-Canvas (& canvas-to-SVG) Parser http://fabricjs.com

LinkageSel

javascript Infinite level linkage select, chainable select, generate <select> element dynamitically, support local, ajax data get http://linkagesel.xiaozhong.biz/index_en.html

ftscroller

FTScroller is a cross-browser Javascript/CSS library to allow touch, mouse or scrollwheel scrolling within specified elements, with pagination, snapping and bouncing support.

toastrjs

Simple javascript toast notifications http://www.toastrjs.com
