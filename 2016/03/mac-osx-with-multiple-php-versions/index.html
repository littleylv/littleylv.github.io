Mac OS X 10.11安装php7

发表于 2016-03-17 更新于 2017-07-10 分类于 OSX

PHP7发布已经几个月了，一直没有去亲身体验。这几天一个关于php7的帖子热传（How Badoo saved one million dollars switching to PHP7），今天决定来体验一下php7的魅力。

由于之前MAC版本一直是OS X 10.10 Yosemite，系统也提示了好久了可以升级OS X 10.11 El Capitan，所以就花了半个多小时升级了系统。最新版强迫症伤不起（：

此次安装php7并非要替换原有的php版本，而是需要保证之前的php55版本能继续使用，而且能方便的切换php版本。对这块的操作并不是很了解，如果完全自己摸索，虽然会有很多乐趣，但可能会走弯路，因此决定从网上查找资料。

爬墙google后找到了一篇非常好的文章，内容详细，步骤清晰。按照步骤一步步操作下来，很顺利的安装了双版本php，且可以用命令一句话切换版本！

php7的module目前还比较少，不过一些通用的基本可以通过sudo brew install php70-xxx来安装，没有的只能通过网上找源码来编译，例如php-memcached目前php7版本还是dev分支。
